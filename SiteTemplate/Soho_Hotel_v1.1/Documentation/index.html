A) HTML Structure - top

This template is a responsive layout with two columns. The main content area has the class "main-content" (two thirds of the page area) and the sidebar area has the class "sidebar" (one third of the page area).



There are 2 primary fonts used in this template "Merriweather" for titles and "Open Sans" for all other text. If you want to change them go to lines 161 and 165 in the "style.css" file. Note both are Google fonts, if you want to change to a different Google font don't forget to import it in the HTML head tags (see line 24/25 of any HTML file in this package)

B) CSS Files and Structure - top

There is one main CSS file which is stored in the "css" directory named "style.css", this contains all the main CSS code. There are 5 other CSS files used for the slideshow, photo lightbox, drop down menu, responsive and colours also found in the "css" directory.

B1) Colours

There are 4 colour schemes packaged with Soho Hotel which can be found in "HTML/css/colours". To change the colour open any HTML file and edit line 19.

C) JavaScript - top

This theme imports 8 Javascript files.

  1. jQuery
  2. jQuery UI
  3. prettyPhoto (photo gallery lightbox)
  4. Superfish (main menu drop down)
  5. Flexslider (homepage slideshow)
  6. Selectivizr (CSS3 for IE6 - 8)
  7. All custom JS code is stored in scripts.js

The only JS file you're likely to need to edit if any is "scripts.js", for example if you want to edit the drop down menu, slideshow or prettyPhoto lightbox settings.
